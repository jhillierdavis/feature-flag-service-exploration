
# Implementation Steps

## Initial setup

'npx create-react-app growth-book-demo-react-app'

'cd growth-book-demo-react-app'

'npm i --save @growthbook/growthbook-react'

'npm install --save react react-dom react-scripts'

'npm audit fix --force'
